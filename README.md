## 功能说明:

服务器配置：端口8080，上下文路径/api
Spring应用名称：async-task-framework-redis
MySQL数据源配置：连接地址、用户名、密码、连接池参数
Redis配置：主机、端口、密码、数据库、超时时间、Lettuce连接池
MyBatis配置：mapper位置、实体类别、驼峰命名、默认执行器、日志
任务调度器配置：拉取间隔100ms、批量大小50
线程池配置：核心8、最大16、队列1000
任务执行配置：默认超时300秒、最大重试3次
Actuator健康检查端点配置

## 关键配置项：
Redis延迟队列使用ZSET存储任务，score为执行时间戳
分布式锁防止任务重复执行
线程池拒绝策略：CallerRunsPolicy防止任务丢失
乐观锁版本号控制并发更新
支持任务执行统计和监控
